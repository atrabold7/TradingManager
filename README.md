Ein effizientes Tool für die Erfassund und Auswertung von Trades (Aktien, Bitcoin, ETF).
Verwendete Sprache: C++ 20. Dieses Projekt befindet sich noch im Aufbau und ist noch nicht 
funktionsfähig. Neue Features werden ständig implementiert.

Features:
Erfassung von Trades
Präzise Berechnung vom Gewinn unter Berücksichtigung von Steuern und Gebühren

Ausblick:
Speicherung der Daten im JSON Format
